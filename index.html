Key Overlay

A customisable on-screen keyboard for mobile browsers. Tap any button to fire real keyboard events into the page.

Safari iOS · Bookmarklet · No install

1 Create the GitHub repo & enable Pages

  1. Create a new GitHub repository (can be public or private).
  2. Upload overlay.js and index.html to the main branch root.
  3. Go to Settings → Pages, set Source to Deploy from a branch, pick main / (root), and save.
  4. After ~60 seconds your file will be live at:
    https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/overlay.js

2 The bookmarklet URL

Replace the two placeholders, then save this entire string as a bookmark URL:

javascript:(function(){var s=document.createElement('script');s.src='https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/overlay.js?t='+Date.now();document.head.appendChild(s);})();

The ?t=… timestamp busts the browser cache so you always get the latest version of the script.

3 Save the bookmarklet on iPhone Safari

  1. Open any webpage in Safari on your iPhone.
  2. Tap the Share button (square with arrow) → Add Bookmark. Tap Save.
  3. Now tap the Bookmarks icon (open book) in the toolbar → find the bookmark you just saved → tap Edit.
  4. Clear the URL field and paste the full javascript:… string from Step 2. Tap Done.
Tip: AirDrop or iMessage the bookmarklet URL from your Mac to your iPhone so you don't have to type it. On Mac, paste it into Notes, then open Notes on iPhone and copy from there.

4 Using the overlay

  1. Navigate to the game or web app you want to control.
  2. Open Bookmarks and tap your bookmarklet. A semi-transparent control bar appears at the top of the screen.
  3. Tap + to add a button. Type the key name (e.g. ArrowUp or w), optionally customise the label, then tap Add.
  4. The overlay enters Edit mode automatically — drag the new button to where you want it on screen.
  5. Tap Done to lock the layout. Buttons are now tap-to-fire.
  6. Your layout is saved per-site in localStorage. Next time you fire the bookmarklet on the same site, your buttons are restored.
If key presses aren't registering: tap the game canvas directly to focus it, then use your overlay buttons. Some games only listen on a focused canvas element.

5 Edit mode

  • Tap Edit in the control bar to enter edit mode. Buttons show a pulsing ring.
  • Drag buttons anywhere on screen to reposition them.
  • Tap the red ✕ on any button to delete it.
  • Tap Done to save the layout and return to play mode.

Shortcuts accepted

You can also type shorthand: up, down, left, right, space, esc, ctrl, enter, backspace.
Single characters like w, a, s, d, 1–0 work as-is.

7 Troubleshooting

  • Overlay doesn't appear — the site may have a strict Content Security Policy that blocks dynamic script injection. Nothing can be done about this from a bookmarklet.
  • Key presses ignored — tap the game canvas or iframe to focus it, then use the overlay buttons.
  • Buttons in wrong position after rotation — enter Edit mode, rearrange, then tap Done to re-save.
  • Want a fresh start on this site — open the browser console and run localStorage.removeItem('keyOverlay_v1_' + location.hostname), then reload the bookmarklet.
